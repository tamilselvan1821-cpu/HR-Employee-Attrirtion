# HR-Employee-Attrirtion
This project analyzes employee attrition using an HR dataset to identify the key factors that influence employees leaving an organization. A Decision Tree classifier is used to build an interpretable machine learning model that helps HR teams understand attrition drivers and make data-driven retention decisions.
